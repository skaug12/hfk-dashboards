동기화

체크 상태는 어디에 저장되나요?

이 브라우저의 localStorage에 자동 저장됩니다.
다른 기기나 브라우저에서는 체크 상태가 공유되지 않습니다.

Apple Reminders / Slack과 동기화하려면

이 대시보드는 정적 웹페이지라, 브라우저에서 직접 Reminders나 Slack에 접근할 수 없습니다.

대신 Claude Code에서 아래 스킬을 실행하면 양방향 동기화됩니다.

/update-dashboard
이 스킬을 실행하면:
1. Google Calendar에서 세션 진행 수를 자동 갱신
2. Apple Reminders ↔ 대시보드 체크리스트 양방향 동기화
3. Slack to-do-log에 미완료 항목 포스팅
4. product notes에서 팀 데이터 최신화

동기화 흐름

웹에서 체크 → /update-dashboard 실행 → Reminders 완료 처리 + Slack 업데이트
Reminders에서 체크 → /update-dashboard 실행 → 웹 대시보드 반영

대시보드 사용법

이 대시보드는 뭔가요?

HFK의 시즌 운영 전체를 한눈에 보는 페이지입니다.
현재 진행 중인 팀, 다음 시즌 기획, 해야 할 일, 일정을 탭별로 확인할 수 있습니다.

탭 구성

  • 개요: KPI 달성률, 시즌 요약 숫자, 바로 해야 할 일 5개
  • 현시즌 운영: 운영 중인 팀을 성장 트랙(코카콜라 5방향)별로 분류 + 매일/주간/월간 운영 체크리스트
  • 다음시즌 기획: 여름시즌 팀 구성 + 기획 체크리스트. 각 항목에 📊 표시가 있으면 데이터 기반 인사이트가 달려 있어요
  • 타임라인: 준비 4주 + 운영 12주를 Gantt 차트로 표시. 가로 스크롤하면 전체 일정이 보이고, 왼쪽 카테고리는 고정됩니다

체크리스트 사용법

  • 항목을 클릭하면 체크/해제됩니다
  • 체크 상태는 이 브라우저에 자동 저장됩니다 (localStorage)
  • 다른 컴퓨터/브라우저에서 열면 체크 상태가 초기화됩니다
  • 매주 수요일에 주간 초기화 버튼을 눌러 주간 루틴을 리셋하세요
  • Apple Reminders, Slack과 동기화하려면 동기화 버튼을 눌러 안내를 확인하세요

성장 트랙 (코카콜라 5방향)

  • Vertical 책임 확장: 더 큰 역할을 준비하는 방향
  • Horizontal 새로운 시각: 다른 분야의 시각을 배우는 방향
  • Enrichment 현재 역할 심화: 지금 하는 일을 더 잘하는 방향
  • Realignment 커리어 전환: 완전히 다른 방향을 설계
  • New Horizons 외부 탐색: 현재 직무 바깥의 성장 기회

타임라인 색상

  • 기획 시즌 준비 업무 (일정, 파트너, 상품)
  • 운영 실행 업무 (채널 생성, 명찰, 시트)
  • 콘텐츠 시즌레터, 포스팅, 인터뷰
  • 모집/홍보 사전등록, 재등록, 일반 모집
  • 인사이트 서베이, 피드백, AAR
  • 세션 그룹A/B 회차 진행
  • 빨간 배경 마감/중요 이벤트
